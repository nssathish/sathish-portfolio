# Sathish Kumar Portfolio

Professional portfolio website for Sathish Kumar.

## Live Site

- GitHub Pages: https://nssathish.github.io/sathish-portfolio/

## Tech Stack

- HTML
- CSS
- JavaScript

## Local Preview

From the project directory:

```bash
python3 -m http.server 8080
```

Then open:

- http://localhost:8080/

## GitHub Pages Setup

Repository settings:

1. Go to `Settings -> Pages`
2. Under **Build and deployment**:
   1. Source: `Deploy from a branch`
   2. Branch: `main`
   3. Folder: `/ (root)`
3. Save

## Key Files

- `index.html` - Main portfolio page
- `resume.html` - Resume viewer page
- `styles.css` - Styling and responsive layout
- `script.js` - Minor client-side behavior
- `Sathish_Kumar_Resume_2026.pdf` - Resume PDF
- `iiit-b-certificate-ds.pdf` - PG DS/AI certificate
- `profile-photo.jpg` - Profile image
